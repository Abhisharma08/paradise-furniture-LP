# **App Name**: Paradise Furnishings

## Core Features:

- Dynamic Header and Navigation: Implement a responsive header featuring the company logo, primary navigation links (Experience, Testimonials, My Biz), and a 'Get a Quote Now' call-to-action.
- Hero Banner Display: Showcase a compelling hero section with the main headline 'Buy Office Chairs in Varanasi – Authorized Dealer of Paradise Furniture', a descriptive subheading, and a 'Download Our Catalogue' button.
- Information and Feature Sections: Present detailed sections such as 'Experience the Paradise Difference', 'Maximize Your Productivity with Ergonomic Seating Solutions', and 'Your Trusted Furniture Partner' with accompanying text, certifications, and benefit highlights.
- Product Series Showcase: Display a grid-based product catalogue with images, titles (e.g., Premium Series, Diamond Series), short descriptions, and 'Enquire Now' buttons for various seating solutions.
- Multi-step Catalogue Request Form: Implement a multi-step form to allow users to request a catalogue, capturing necessary contact information as depicted in the original layout.
- Custom Design Inquiry Form: Provide a mechanism (via button leading to a form) for users to inquire about custom furniture designs, aligning with the 'Need a Custom Design?' section.
- HubSpot API Submission: Integrate with the HubSpot API to securely send data captured from the 'Catalogue Request Form' and 'Custom Design Inquiry Form' for lead management.

## Style Guidelines:

- Primary color: Deep, professional red (#B32D2D), derived from/complementing the company logo. This saturated hue evokes confidence and professionalism, providing a strong anchor for calls to action and important highlights, suitable for the furniture industry.
- Background color: Very light, subtle reddish off-white (#F8F2F2), chosen to complement the primary logo color. This desaturated variant of the primary hue creates a clean and expansive backdrop, ensuring readability and a premium feel in a light scheme.
- Accent color: Deep plum (#7A3D6B), selected to harmonize with the logo's palette. Analogous to the primary red, this sophisticated and muted tone provides an alternative highlight color that contrasts in saturation and brightness, adding depth without overpowering the primary palette.
- Body and headline font: 'Inter' (sans-serif) for its clean, modern, and objective appearance, ensuring excellent readability for all content from headlines to detailed descriptions.
- Utilize clean, minimal line-art or outline icons, consistent with the existing page's use of simple visual cues for benefits and certifications, maintaining a contemporary and uncluttered look.
- Employ a modular and responsive layout with distinct, well-spaced sections. The design should emphasize clarity and product showcasing through grid systems for catalogue items and a streamlined presentation for informational blocks, similar to the original content.
- Incorporate subtle hover effects on interactive elements like buttons and product cards, alongside smooth transitions for navigation between multi-step form sections, to provide a polished and intuitive user experience.
